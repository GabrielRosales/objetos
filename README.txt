Para iniciar el programa y ejecutarlo siga los siguientes pasos:
Nota: Es necesario que tenga instalada la paquetería de Java SE de lo contrario el programa no podrá ser ejecutado. Para descargar la paquetería puede ir al siguiente enlace: https://goo.gl/8jFJ8J. Si el enlace no funciona visite la página: https://www.oracle.com/technetwork/es/java/javase/downloads/index.html
Además, será necesario tener instalado NetBeans en la versión actualizada.

Primera alternativa:
1.- Descargar o clonar el proyecto del repositorio.
2.- Guardar la carpeta del proyecto en la carpeta generada después de la instalación de NetBeans
3.- Iniciar NetBeans, abrir la pestaña File, seleccionar Open Project y abrir el proyecto con el nombre de objetos
4.- Después de que el proyecto se cargue por completo, puede presionar F6 para correr el proyecto.
5.- Sigua las instrucciones que le muestre el programa.
